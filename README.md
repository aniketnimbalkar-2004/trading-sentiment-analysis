# Trading Behavior Analysis using Market Sentiment

##  Objective
Analyze trader performance based on Fear vs Greed sentiment.

##  Dataset
- Trading dataset
- Sentiment dataset

##  Key Metrics
- PnL
- Win Rate
- Leverage
- Trade Size
- Long/Short Ratio

##  Analysis
Compared trader behavior across Fear vs Greed conditions.

##  Key Insights
- Greed → Higher risk & volatility
- Fear → Conservative trading
- High leverage → Inconsistent returns

##  Strategies
- Reduce leverage during Fear phases
- Controlled trading during Greed phases
